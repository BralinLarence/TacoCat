                            function getValue() {
                                document.getElementById("alert").classList.add("invisible");

                                let userString = document.getElementById("userString").value;

                            //LOGIC FUNCTION(S)
                            //Check string for palindrome
                            function isPalindrome(_userString) {
                            
                                _userString = _userString.toLowerCase();
                            
                                //remove spaces in special characters
                                let regex =/[^a-z0-9]/gi;
                                _userString = _userString.replace(regex, "");

                                //reverse the string
                                let reversedString = [];
                                let returnObj = {};

                                for (let index = _userString.length - 1; index >= 0; index--) {
                                reversedString += _userString[index];
                                }

                                //Check if a Palindrome is entered.
                                if(reversedString == _userString)
                                {
                                    returnObj.msg = "Well Done! You entered a Palindrome!";
                                    returnObj.isTrue = true;
                                }
                                else{
                                    returnObj.msg = "Sorry! you did not enter a Palindrome!";
                                    returnObj.isTrue = false;
                                }

                                returnObj.reversed = reversedString;

                                return returnObj
                            }

                            //VIEW FUNCTION(S)
                            //Display the reversed string to the user
                            function displayMessage(returnObj) {

                                if(returnObj.isTrue)
                                {
                                    document.getElementById("alert").classList.remove("invisible", "alert-danger");
                                    document.getElementById("alert").classList.add("alert-success");
                                    document.getElementById("message").classList.remove("alert-danger");
                                    document.getElementById("message").classList.add("alert-success");
                                    document.getElementById("alertHeader").innerHTML = returnObj.msg;
                                    document.getElementById("message").innerHTML = `Your phrase reversed is: ${returnObj.reversed}`;

Introduction

This code uses the Model View Controller (MVC) architectural pattern. There are three separate components in this pattern such as Model, View, Controllers as shown in the name of the design pattern . Each function has its own responsibilities which makes it easier to maintain and update the code base through it's life cycle.

getValue

The controller function getValues calls the input value from the userString variable. Next, it calls the isPalindrome function to check if the userString is a palindrome and returns the value to the variable returnObj. Finally, the returnObj is passed into the function displayMessage as an argument.

isPalindrome

Inside the isPalindrome function we set the value of the userString variable to lower case. Then call a function on the userString variable called replace(). We set the parameters to a regex and empty string argument that determines if any whitespace or special characters are in the string, removes and replaces it with the empty string. Next, a for loop is used to reverse the string. An if/else statement then compares the reversed string with the original. If the string is a Palindrome then a message will state that and set the returnObj property isTrue to true, otherwise a message will state that it is not and the returnObj isTrue property will return false. Lastly, we assign the reversed string to the reversed property of the object and return it.

displayMessage

The displayMessage function will show a green success alert window if the property of the object returnObj is true, meaning the input string is a Palindrome. If not, then it will display a red alert window stating that the string is not a Palindrome.
